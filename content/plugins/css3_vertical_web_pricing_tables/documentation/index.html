CSS3 Vertical Web Pricing Tables for WordPress
About

CSS3 Vertical Web Pricing Tables for WordPress is a WordPress plugin with following features:

  • pure CSS + HTML (no javascript)
  • 10 color versions
  • animated hover states
  • configurable width
  • unlimited number of rows
  • unlimited number of features for each row
  • font-face fonts
  • PSD source included
  • intuitive admin panel
  • implementation with simple shortcode

You can check sample pricing tables on this screencast.

Installation

Installation requires just few simple steps:

  • unpack css3-verticals-web-pricing-tables-for-wodpress.zip archive
  • log into your WordPress administration panel
  • open the "Plugins" submenu
  • click "Add new" button
  • click "Upload" link
  • select css3_vertical_web_pricing_tables.zip archive from unpacked folder (css3-verticals-web-pricing-tables-for-wodpress folder)
  • click "Install now"
  • after installation process click "Activate" link
Implementation

You can configure the pricing table using administration panel. After installation you will find new subsection (CSS3 Vertical Web Pricing Tables) under "Settings" submenu. Please open it and follow below steps:

  • choose one of 10 predefined sample configurations from "Choose shortcode id" list...
  • ...or create your own new configuration by simply putting its name/identifier into "Or create new shortcode id" textfield
  • choose style/color from 10 available
  • define table header and features section header titles
  • define number of table rows
  • for each row define header and number of features
  • fill features fields
  • put "buy" button html code into "Row n button" textfield. Sample html: 
    <a href='#' class='vt-button'>Buy now</a>
  • optionally define paddings, arrow sizes for each row
  • optionally define vartical table width
  • click "Preview" button to see how the table will look on your page
  • click "Save Options" button to save the configuration
  • after saving operation you'll see the notice with shortcode. For example 
    [css3_vertical_table id='sample_identifier']
    Just simply copy it and paste it on your page or post
  • you can come back to your configuration and edit it in any time, just choose it from "Choose shortcode id" list
